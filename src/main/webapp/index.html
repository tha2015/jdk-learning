Call Java Methods with String Input/Output:

How it works:
  • Java methods marked with @JSExport are callable from JavaScript
  • Type text in the input field and click buttons to call Java methods
  • JavaScript passes strings to Java methods and receives string results
  • You can also call from console: processString("test")
